열림 (Opening) : 침식 후 팽창, 깎아서 노이즈 제거 후 살 찌움¶

dilate(erode(image)))

import cv2
import numpy as np
kernel = np.ones((3,3), dtype=np.uint8)

img = cv2.imread('erode.png', cv2.IMREAD_GRAYSCALE)

erode = cv2.erode(img, kernel, iterations=4)
dilate = cv2.dilate(erode, kernel, iterations=4)

cv2.imshow('img', img)
cv2.imshow('erode', erode)
cv2.imshow('dilate', dilate)
cv2. waitKey(0)
cv2.destroyAllWindows()

침식을 4회 반복 -> 노이즈 제거 -> 글자 얇아짐 -> 팽창을 4회 반복 -> 원본 글자 크기 (노이즈제거됨)

import cv2
import numpy as np
kernel = np.ones((3,3), dtype=np.uint8)

img = cv2.imread('dilate.png', cv2.IMREAD_GRAYSCALE)

dilate = cv2.dilate(img, kernel, iterations=3)
erode = cv2.erode(dilate, kernel, iterations=3)


cv2.imshow('img', img)
cv2.imshow('erode', erode)
cv2.imshow('dilate', dilate)
cv2. waitKey(0)
cv2.destroyAllWindows()

팽창 3회 반복 (글자 외곽에 하얀색 픽셀 채워짐) -> 글자 엄청 두꺼워짐(동시에 구멍 메워짐) -> 침식 3회 반복 (하얀색 픽셀을 까만색으로 바꿈) -> 원본 글자 크기 (구멍 메워짐)
