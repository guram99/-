직선¶

직선의 종류 (line type)

  1. cv2.LINE_4 : 상하좌우 4 방향으로 연결된 선
  2. cv2.LINE_8 : 대각선을 포함한 8 방향으로 연결된 선 (기본값)
  3. cv2.LINE_AA : 부드러운 선 (anti-aliasing)
In [12]:
import cv2
import numpy as np

img = np.zeros((480,640,3), dtype=np.uint8)

COLOR = (0,255,255) #BGR : Yellow
THICKNESS = 3 # 두께

cv2.line(img, (50,100), (400,50), COLOR, THICKNESS, cv2.LINE_8)
cv2.line(img, (50,200), (400,150), COLOR, THICKNESS, cv2.LINE_4)
cv2.line(img, (50,300), (400,250), COLOR, THICKNESS, cv2.LINE_AA)
# 그릴 위치, 시작 점, 끝 점, 색깔, 두께, 선 종류

cv2.imshow('img',img)
cv2.waitKey(0)
cv2.destroyAllWindows()

라인 8은 대각선도 그릴 수 있어서 픽셀이 끊어지는 것을 알 수 있음.
라인 4는 상하좌우만 그릴 수 있어서 픽셀이 계속 이어짐을 알 수 있음.
라인 AA는 8,4 보다 선이 부드럽게 이어짐.

import cv2
import numpy as np

img = np.zeros((480,640,3), dtype=np.uint8)

COLOR = (0,0,255) #BGR : 빨간색
THICKNESS = 3 # 두께

pts1 = np.array([[100,100],[200,100],[100,200]])
pts2 = np.array([[200,100],[300,100],[300,200]])

#cv2.polylines(img, [pts1], True, COLOR, THICKNESS, cv2.LINE_AA)
#cv2.polylines(img, [pts2], True, COLOR, THICKNESS, cv2.LINE_AA)

cv2.polylines(img, [pts1, pts2], True, COLOR, THICKNESS, cv2.LINE_AA) # 속이 빈 다각형
# 그릴 위치, 그릴 좌표들, 닫힘 여부, 색깔, 두께, 선 종류

# polylines를 통해 다각형을 그릴 때에는 두 번째 들어가는 값의 데이터들을 내가 그리고 싶은 지점을 정의한 다음 리스트 형태로 넣어주면 거기 있는 모든 다각형을 그려줌.
# True : 닫힌 도형 / False : 열린 도형(처음 점과 끝 점 연결 x)

pts3 = np.array([[[100,300],[200,300],[100,400]],[[200,300],[300,300],[300,400]]]) # pts3 자체를 리스트 처리
cv2.fillPoly(img, pts3, COLOR, cv2.LINE_AA) # 속이 찬 다각형
# 그릴 위치, 그릴 좌표들, 색깔, 선 종류

cv2.imshow('img',img)
cv2.waitKey(0)
cv2.destroyAllWindows()
